== Installation ==
1. Upload the folder "hide-wp-upgrade-message" to "/wp-content/plugins/" path.
2. Activate the plugin through the "Plugins" menu in WordPress backend
3. After activating the plugin, upgrade message will automatically disappeared.
4. Enjoy!
